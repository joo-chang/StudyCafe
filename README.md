## 스터디카페 무인 시스템
-------------
* ## 개요
> 주로 스터디카페 입구에 배치되어있는 기계로 회원관리, 좌석관리, 요금관리 등 여러 가지 기능을 사람 대신 충족시켜준다. 관리자와 회원을 계정을 통해 구별하여 서로 다른 기능을 구현한다. 또, 우리 팀원들은 기존의 기능에 채팅기능을 추가하여 관리자와 회원의 간단한 의사소통 기능도 더했다.

-------------
* ## 개발 환경 및 사용 기술
* 운영 체제 - Windows
* 개발 툴 - Eclipse
* 개발 언어 - JAVA
* Database - Mysql
